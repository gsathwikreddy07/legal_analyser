# Legal RAG Contract Risk Analyzer

## Project Overview
An AI-powered system that analyzes legal contracts and identifies risky clauses using Retrieval-Augmented Generation (RAG).

## Objectives
- Reduce manual contract review time
- Improve consistency and explainability
- Support scalable legal document analysis

## Core Features
1. Upload contract files (PDF/TXT)
2. Extract and preprocess text
3. Chunk long documents
4. Generate embeddings
5. Store vectors in FAISS
6. Retrieve relevant clauses
7. Analyze risk using an LLM
8. Show summary, citations, and recommendations

## Technology Stack
- Python
- Sentence Transformers
- FAISS
- Transformers / FLAN-T5
- Gradio / Streamlit

## Dataset
CUAD (Contract Understanding Atticus Dataset)

## Workflow
1. Upload contract
2. Extract text
3. Chunk text
4. Create embeddings
5. Retrieve relevant clauses
6. Generate risk analysis
7. Show final report

## Disclaimer
This tool assists review workflows and is not legal advice.
